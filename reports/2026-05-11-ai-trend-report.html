市場訊號

01值得追

AI 算力與晶片供應鏈：由採購競賽轉向資本與地緣風險管理

發生什麼

ByteDance 加大 AI 投資、OpenAI / Broadcom 自研晶片需要大型買家承諾、NVIDIA 以股權交易鎖定生態，顯示算力競爭已由「買 GPU」變成長約、股權、保底訂單、政策與能源約束的組合。

誰收錢

短期收現金較快的是硬瓶頸供應商：GPU、HBM、先進封裝、networking、電力設備、冷卻與 data center site。

誰埋單 / 承擔風險

Cloud、GPU cloud、data center operator、模型公司與自研晶片項目承擔 capex、折舊、utilization 與供應鏈地緣風險。訂單、融資、股權投資與保底採購不是同一件事，要分開看。

真正定價權在哪

短期定價權在硬瓶頸與交付可靠性；中期要看供給改善、需求真實度與長約覆蓋率。若 AI workload 不足，高 capex、低 utilization、沒有長約支撐的一層最易受傷。

接下來看什麼

大型雲端 capex 指引、NVIDIA 投資披露、ByteDance 晶片供應商名單、OpenAI / Microsoft 採購承諾、亞洲數據中心電力審批。

02值得追

Agent 安全：由「答錯」升級為「自主造成損害」

發生什麼

多個安全評測與媒體訊號顯示，agent 風險正由文字輸出錯誤，升級為工具操作、權限、沙箱、回滾與責任鏈問題。

誰收錢

Agent 安全、審計、沙箱、身份權限、事故回滾、觀測與合規工具會較早受益。

誰埋單 / 承擔風險

想把 agent 放入真實工作流的企業、雲端平台與 agent 產品商要先埋單：安全測試、權限設計、行動日誌、保險與法務審批都會變成固定成本。

真正定價權在哪

定價權不在「模型會答題」，而在誰能證明可控、可追溯、可回滾。裸模型與缺乏治理層的 agent app 容易被壓價。

接下來看什麼

模型公司 agent 安全白皮書、企業版權限政策、第三方評測、保險條款、雲端平台 agent 審計記錄。

03值得追

高成本模型：AI 應用要重新計算單位經濟

發生什麼

高能力模型價格上升，同時開發者平台開始討論 agentic workflow 的 token 效率，市場焦點由「模型是否更強」轉向「每次任務成本是否可控」。

誰收錢

Frontier model API、企業合約平台、模型路由與成本觀測工具有機會收現金。

誰埋單 / 承擔風險

AI SaaS、agent app 與用量快速增長但客單價不足的產品要承擔推理成本、重試成本、長上下文成本與毛利壓力。

真正定價權在哪

高價值 reasoning / coding / enterprise-critical 任務仍可支付 premium；摘要、分類、客服等普通任務會被低價模型、開源模型或本地模型替代。真正壁壘是模型路由、快取、檢索與任務分層。

接下來看什麼

主要模型 API 價格、企業折扣、agent 平台成本面板、開源替代、用量限制調整、AI SaaS 毛利或用量成本披露。

次要觀察

語音 AI 進入辦公入口

市場感：入口價值在 workflow 和分發，不在語音 demo。OS、辦公套件、耳機、會議工具與輸入法較可能掌握入口；企業 IT、客服中心與本地化應用承擔私隱、噪音與錯字修正成本。

接下來看：企業版定價、離線模式、耳機/OS 合作、多語言準確率、辦公室常開麥克風政策。

AI 廣告與短內容工具

市場感：AI 降低創意初稿成本，但未必降低 CAC。真正有定價權的是掌握分發、測試迴路與轉換資料的平台；只賣素材製作的代理商會被壓價。

接下來看：Google Ads AI 創意產品化、短片 feed 轉換數據、品牌安全規則、CPM/CAC 是否下降。

本地 AI / 硬件信任

市場感：本地 AI 是雲端 AI 的反向壓力測試。收錢機會在裝置端晶片、私有部署、企業資料治理與安全工具；成本在硬件更新、維護與 IT 支援。

接下來看：瀏覽器/OS 本地模型 API、企業私有部署需求、硬件認證政策爭議、開源模型下載量。
